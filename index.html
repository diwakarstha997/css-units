CSS UNITS

px (Pixels)

Pixels are the most common unit used in web design. It is an absolute unit that represents a single pixel on the screen.

Box with height of 50px

mm (Millimeters)

The mm unit represents physical millimeters. It can be used for very fine control in physical measurements.

Box with font-size of 10mm

in (Inches)

The in unit represents inches. 1 inch equals 2.54 centimeters and is typically used for print.

Box with font-size of 1 inch

cm (Centimeters)

The cm unit represents physical centimeters. It's useful for print-based designs but can also be used for web pages.

Box with font-size of 4cm

pt (Points)

The pt unit is commonly used in print media. 1 point is equal to 1/72 of an inch.

Box with font-size of 12pt

% (Percentage)

The % unit is relative to the parent element's size. This unit is commonly used for responsive layouts.

Box with width 50% of parent

vw (Viewport Width)

The vw unit is relative to the viewport's width. 1vw is equal to 1% of the viewport's width. It is very useful for making elements scale according to the screen size.

Box with width 50vw (50% of the viewport width)

vh (Viewport Height)

The vh unit is relative to the viewport's height. 1vh equals 1% of the viewport's height.

Box with height 30vh (30% of the viewport height)

em

The em unit is relative to the font size of its closest parent element. It allows for scalable layouts.

Box with 1.5em font-size (relative to parent)

rem

The rem unit is relative to the root element's font size. It provides consistency across the page.

Box with 2rem font-size (relative to root element)
